Chrome Extension Tutorial

Google Chrome เป็น Web browser ตัวหนึ่งที่ได้รับความนิยมแพร่หลายในปัจจุบัน ด้วยความสวยงามของโปรแกรม ความเร็วในการโหลดและภาพรวมการใช้งาน อีกทั้งยังใช้งานง่าย และมี “ส่วนเสริม” ที่เรียกว่า Extension ให้ได้โหลดมาติดตั้งตามความต้องการอีกด้วย

Installation


1. เปิดกูเกิลโครม (Google Chrome)



2. เข้าหน้าร้านสำหรับส่วนขยาย Chrome Web Store



3. กดเลือกส่วนขยาย (extension) ที่สนใจ



4. กดปุ่ม ADD TO CHROME ก็จะเริ่มทำการติดตั้งให้ทันที



5. เมื่อทำการติดตั้งส่วนขยายแล้ว จะมีการสร้างปุ่มลัดให้อัตโนมัติ เพื่อให้ผู้ใช้สามารถกดใช้งานได้ในทันที

Panic Button

บางครั้งผู้ใช้อินเทอร์เน็ตอย่างเราๆก็ไม่ได้อยากเปิดเผยสิ่งที่เรากำลังสนใจอยู่เสมอไป หลายๆครั้งอยู่ดีๆก็มีคนมาจากข้างหลังซึ่งกำลังจะเห็นสิ่งที่คุณทำอยู่
